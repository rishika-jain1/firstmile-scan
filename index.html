How to set up Google Sheets sync
1. Open Google Sheets → Extensions → Apps Script
2. Paste the Apps Script code (see below)
3. Click Deploy → New deployment → Web app
4. Set Execute as: Me, Access: Anyone
5. Copy the URL and paste it above

Apps Script code:
function doPost(e){var s=SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();var d=JSON.parse(e.postData.contents);if(s.getLastRow()===0)s.appendRow(['Timestamp','AWB','Status','L(cm)','W(cm)','H(cm)','Vol.Wt(kg)','Act.Wt(kg)','Final.Wt(kg)','Volume(m3)','Category','Address','FE ID','Van?','Confidence','Remarks']);s.appendRow([d.ts,d.awb,d.status,d.l,d.w,d.h,d.volWt,d.actWt,d.finalWt,d.vol,d.category,d.address,d.fe,d.van?'Yes':'No',d.conf,d.remarks]);return ContentService.createTextOutput('ok');}
